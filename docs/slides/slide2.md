# 幻灯片2：自我介绍页

## 基本信息
- **幻灯片编号**: 2
- **标题**: 分享人介绍
- **布局类型**: 内容页
- **预计展示时间**: 45秒

## 内容设计

### 标题区域
- **标题文本**: 分享人介绍
- **标题样式**: 大号像素字体，主色
- **下划线**: 4px像素下划线，辅色

### 内容区域
采用项目符号列表布局，每项一个8-bit风格卡片：

1. **身份标识**
   - **图标**: 像素化人物头像
   - **文本**: "Rex（我自己）"
   - **说明**: 主演讲者标识

2. **职业背景**
   - **图标**: 像素化剧本/教育图标
   - **文本**: "儿童教育领域创业者（剧本策划方向）"
   - **说明**: 专业领域说明

3. **工作核心**
   - **图标**: 像素化文字/键盘图标
   - **文本**: "工作核心：与文字打交道"
   - **说明**: 核心技能描述

4. **具体工作内容** (子项目)
   - **图标**: 像素化清单图标
   - **文本**:
     - "写大纲"
     - "打磨剧本"
     - "润色对白"
   - **说明**: 日常工作分解

## 8-bit设计规范

### 布局结构
- **整体布局**: 垂直列表，每项占一行
- **项目间距**: 24px (6个像素单位)
- **内容宽度**: 80%容器宽度

### 卡片设计
每个项目使用8-bit风格卡片：
- **背景**: 轻微渐变背景 (`#F7CAC9` → `#FFFFFF`)
- **边框**: 4px实线边框 (`#2A3B47`)
- **内边距**: 16px (4个像素单位)
- **图标尺寸**: 32×32像素
- **文字对齐**: 图标左对齐，文字右对齐

### 颜色方案
- **标题色**: `#6B5B95` (紫色调)
- **下划线色**: `#88B04B` (绿色调)
- **卡片背景**: 渐变 `#F7CAC9` → `#FFFFFF`
- **文字色**: `#2A3B47` (深蓝灰)
- **图标色**: `#FF6F61` (珊瑚色)

### 像素细节
- **边框像素**: 4px方形像素
- **阴影**: 卡片下方4px偏移阴影
- **圆角**: 无圆角，保持像素风格
- **图标风格**: 16×16像素放大到32×32，保持像素感

## 交互需求
- **键盘导航**: 左右箭头键切换
- **鼠标点击**: 点击卡片可轻微放大效果
- **焦点状态**: 当前项目有高亮边框

## 视觉增强
1. **像素化背景纹理**: 轻微网格纹理
2. **动态效果**: 进入时卡片逐个弹出
3. **图标动画**: 悬停时图标轻微晃动
